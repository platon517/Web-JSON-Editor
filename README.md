Небольшой редактор json файлов, визуализирующий данные.
#### Используемые технологии:
- [React](https://reactjs.org/)
- [Webpack](https://webpack.js.org/)
- [Babel](https://babeljs.io/)
- [Less](http://lesscss.org/)
